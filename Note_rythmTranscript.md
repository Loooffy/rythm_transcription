#### 狀況
使用者有節奏地按鍵盤，按完之後希望看到譜上記下剛才按下的節奏

#### 程式任務
輸入間歇性的keypress，將之格式化成節奏，然後輸出此節奏的樂譜

#### 工作環境與流程
1. text editor: atom
2. language: Python
3. Note

#### 設計策略
1. 計時：紀錄每次按下按鍵的時間，存到list中
2. 最小單位：找出list中時間相隔最小的數值min
  1. 時間間隔：each T(n+1)-T(n)
  2. 找出最小的T(n-1)-T(n)  
3. 正規化：用min除所有的時間間隔，存到另一個list中
4. 找出小節
5. 輸出成譜
